# src_slint/src/core/cloud

`cloud.rs` のテスト専用サブモジュール。

## 含まれるファイル

| ファイル | 役割 |
|---------|------|
| `cloud_tests.rs` | `merge_into_local`のHARD_RESETマーカー処理（全データ削除・データ圧縮のような上書き系操作が他端末に正しく伝播するか）の単体テスト。`#[cfg(test)]` のみでビルドされる |

## 依存関係

- 依存する（import する）フォルダ：なし（`super::*` で親モジュール `cloud.rs` の非公開関数を参照するのみ）
- 依存される（import される）フォルダ：なし（テストからのみ使われる）

## 注意

テストの入力データはすべて架空の日付（`2024-01-01`〜）を使う。
